# Spam Invaders

A browser-based arcade game built for the Wix Emails team booth at an engineering conference.

Players control a firewall ship and must distinguish between three types of falling emails: shoot the spam, let legitimate emails pass safely, and catch VIP messages for bonus points. The game gets progressively harder with each level, and a combo system rewards quick reactions. When all lives are lost, players enter their name for a top-10 leaderboard — perfect for friendly competition at the booth.

Open `index.html` in a browser to play. No install or build step needed.
